JavaScript functions practice

Open the web console and try these functions work well or not.

isAtom(Arg) -> boolean

Returns true if Arg is string or number, otherwise false.

引数が文字か数字の時にtrueを返し、その他はfalseを返します。

isAtom("Hello");//returns true

isAtom(123);//returns true

isAtom([1, 2, 3]);//returns false

isAtom();//returns false

isLat(Arg) -> boolean

Returns true if Arg is list which is contained by Atom, otherwise false.

引数がAtomからなる配列であったらtrueを、そうでなければfalseを返します。

isLat([1, 2, 3]);//returns true

isLat(["This", "is", "a", "Pen"]);//returns true

isLat([]);//returns true

isLat([[]]);//returns false

isLat("Hello");//returns false

isLat(123);//returns false

isMember(Arg1, Arg2) -> boolean

Arg1 :: Atom, Arg2 :: Lat

Returns true if Arg1(Lat) contains Arg1(Atom), otherwise false.

第二引数(Lat)が第一引数(Atom)を含んでいたらtrueを、その他はfalseを返します。

isMember(1, [1, 2, 3]);//returns true

isMember("Pen", ["This", "is", "a", "Pen"]);//returns true

isMember(1, []);//returns false

rember(Arg1, Arg2) -> term()

Arg1 :: Atom, Arg2 :: Lat

Returns the list which the first argument is removed from second argument.

第二引数(Lat)から第一引数(Atom)を取り除いたリストを返します。

rember(1, [1, 2, 3]);//returns [2, 3]

rember("Pen", ["This", "is", "a", "Pen"]);//returns ["This", "is", "a"]

rember(1, []);//returns []
